# Transit-Typing

## Description
This is a small application to help you writing texts without duplicate words. 

## Background
This work is inspired by the novel [Schleifen](https://www.hanser-literaturverlage.de/buch/elias-hirschl-schleifen-9783552075887-t-5834) by Elias Hirschl in which the mathematician and author Otto Mandel writes a book called *Transit* in which each word only appears once.
